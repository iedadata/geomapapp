Create your GeoMapApp Work Folder

Your GeoMapApp work folder will be used to save:
Preferences.
Custom color palettes.
Locations you like to visit.

Use the File Chooser on the right to create the GeoMapApp work folder
(by selecting the "New Folder" icon in the upper right corner).
If your GeoMapApp work folder already exists, select it.
Press "OK".

If you do not want a GeoMapApp folder, press CANCEL.
